3SB04-Web App Framework (Flask)

นายณัฐพงค์ ปิติศรีพัฒนรากุล

ขอขอบคุณแหล่งข้อมูลที่ใช้ประกอบเป็นเนื้อหาเว็บไซต์ จาก NewMovieBlog

คุณสมบัติของ Flask และ Python ที่ใช้งาน

ฟังก์ชันพื้นฐานของ Flask
  • การนำเข้า Flask: จาก flask การนำเข้า Flask, render_template, request, jsonify - การนำเข้าโมดูลที่จำเป็นจาก Flask
  • การสร้างแอป: app = Flask(__name__) - การสร้างอินสแตนซ์แอปพลิเคชัน Flask
  • การนำเข้าสื่อโดยใช้: url_for() - โค้ด HTML ที่ใช้ในเทมเพลตของ Flask เพื่อแสดงภาพจากแหล่งอ้างอิง
Routes and Views
  • การกำหนดเส้นทาง: @app.route() - การใช้ @app.route เพื่อกำหนดเส้นทาง
Rendering HTML Templates
  • การจัดการการส่งแบบฟอร์ม: render_template() - การแสดงผลเทมเพลต HTML โดยใช้เครื่องมือสร้างเทมเพลต Jinja2
  • การดึงข้อมูลแบบฟอร์ม: request.form.get() - การเข้าถึงข้อมูลแบบฟอร์มที่ผู้ใช้ส่งผ่านวิธี POST
  • การประมวลผลข้อมูลแบบฟอร์ม: ฟังก์ชันเช่น calculate_score() ประมวลผลข้อมูลแบบฟอร์มที่ได้รับจากผู้ใช้
Templates Inheritance
  • การ Reuse Components: extends() - การกำหนดเค้าโครงหรือ Layout ของหน้าเว็บให้มีลักษณะเหมือนกันสำหรับการใช้ในหน้าอื่น ๆ
Returning Responses
  • การ Return JSON Response: jsonify () - การส่งคืน การตอบสนอง JSON จากเส้นทาง Flask
Fetching Data from the Backend (AJAX/Fetch)
  • การส่งข้อมูลไปยังแบ็กเอนด์: fetch('/calculate_score', {...}) - การใช้ Fetch API ของ JavaScript เพื่อส่งข้อมูลแบบฟอร์มไปยังเซิร์ฟเวอร์เพื่อการประมวลผล
เครื่องมือและฟังก์ชันอื่น ๆ ที่นำมาใช้
  • Bootstrap
  • JavaScript
